# GPT Text Generation Project

A PyTorch implementation of a GPT-like transformer model for text generation tasks.

## Project Structure
├── config.json # Configuration file for model, training and evaluation

├── dataset.py # Dataset class for text data loading and preprocessing

├── model.py # GPT model architecture implementation

├── pretrain.py # Training script for the GPT model

└── eval.py # Interactive inference script for text generation
